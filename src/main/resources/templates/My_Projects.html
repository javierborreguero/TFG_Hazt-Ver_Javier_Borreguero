Mis proyectos

Hundir la flota

Java - Sockets

El clásico juego de la batalla naval. Al comenzar cada jugador distribibuye sus barcos a lo largo de su tablero. Con todos los barcos bien colocados, comienza una serie de rondas en las que cada jugador debe derribar todos los barcos de su contrincante. Gana el primer en destruir toda la flota rival. Este proyecto está desarrollado con socket stream (TCP), que es un protocolo orientado a conexión, que permite la comunicación entre los distintos jugadores de manera continua.

Chat

Java - Sockets

En esta ocasión traigo un chat. En el que dos usuarios intercambian diferentes mensajes. Desarrollado con socket datagram (UDP), sevicios de transporte de información sin conexión.

Gestor de vuelos

Html - JavaScript - PHP

Adentrate en el mundo de las aerolíneas. Con esta aplicación, podrás convertirte en un controlador aéreo y controlar todos los vuelos de una aerolínea, gestionar la compra de billeteset et. Con este Gestor de vuelos, aprenderás a desarrollarte en el mundo del fron-end y del back. Desarrollado con Html, Css, JavaScript y PHP.

App Clima

Kotlin - SQLite - Volley - Coroutines

Accede a tu ciudad y consulta su información metereológica. Con App Clima ya no te pillará desprevenido si llueve o hace frío. Ya que sabrás la temperatura antes de salir de casa. Si eres nuevo en la programación para dispositivos Android y quieres aprender el control de usuario mediante persistencia, como acceder a un servicio de internet o a un API, aprenderás a hacerlo mediante "Volley". También usarás SQLite como base de datos para almacenar toda la información.
